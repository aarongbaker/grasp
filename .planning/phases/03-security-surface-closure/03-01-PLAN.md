<threat_model>
## Trust Boundaries

- `POST /sessions` is an authenticated entry point that can consume worker and LLM capacity.
- Slowapi enforcement depends on the chosen key function and limiter wiring matching the route’s auth contract.

## STRIDE Register

| threat_id | category | component | disposition | mitigation_plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Denial of Service | `create_session()` authenticated path | mitigate | Key the limiter by authenticated user identity and enforce the `10/minute` ceiling with route-level regressions. |
| T-03-02 | Spoofing | unauthenticated fallback path | mitigate | Use remote-address fallback only when no authenticated user key is available and hold that fallback to `5/minute`. |
</threat_model>
